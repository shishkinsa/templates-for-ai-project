## 🎯 Описание

Система предоставляет REST API `/api/v1/examples` для демонстрации сквозного сценария: OpenAPI → UseCases → PostgreSQL → SPA.

## ✅ Критерии приемки

```gherkin
Функционал: Примеры сущностей
  Сценарий: Создание и чтение
    Допустим API доступен
    Когда клиент POST /api/v1/examples с телом {"name":"Test"}
    Тогда ответ 201 и тело содержит id и name
    И GET /api/v1/examples/{id} возвращает ту же сущность
```
